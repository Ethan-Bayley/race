Step 2: Import on the new site with query parameter

Alternatively, migrate directly in-browser using URL redirect:

This encodes your localStorage and sends you to https://bookshelf2026.github.io/race?import=....

New site import (for implementation)

const params = new URLSearchParams(window.location.search);
const data = params.get('import');
if (data) {
  try {
    const decoded = atob(decodeURIComponent(data));
    const obj = JSON.parse(decoded);
    Object.entries(obj).forEach(([k,v]) => localStorage.setItem(k,v));
    alert('Data imported successfully!');
  } catch (err) {
    console.error('Import failed', err);
    alert('Import failed: '+err.message);
  }
}
